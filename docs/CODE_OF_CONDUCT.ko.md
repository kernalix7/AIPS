# 행동 강령

[English](../CODE_OF_CONDUCT.md) | **한국어**

## 우리의 약속

나이, 신체 조건, 장애, 민족, 성 정체성, 경험 수준, 국적, 외모, 인종, 종교 또는 성적 정체성과 관계없이 모든 사람에게 괴롭힘 없는 참여 환경을 제공하기로 약속합니다.

## 기준

긍정적인 환경에 기여하는 행동의 예:
- 환영하고 포용적인 언어 사용
- 다른 관점 존중
- 건설적 비판 수용
- 커뮤니티에 최선인 것에 집중

용납되지 않는 행동의 예:
- 괴롭힘, 트롤링, 모욕적 발언
- 공개·비공개 괴롭힘
- 타인의 사생활 정보 공개
- 부적절하다고 합리적으로 간주될 수 있는 기타 행동

## 시행

학대 행위 사례는 [GitHub Security Advisories](https://github.com/kernalix7/AIPS/security/advisories/new)를 통해 신고하실 수 있습니다.

이 행동 강령은 [Contributor Covenant](https://www.contributor-covenant.org/) v2.1을 따릅니다.
